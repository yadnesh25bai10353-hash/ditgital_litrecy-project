Project Overview
This project is part of the Digital Literacy course. The objective is to develop essential digital skills required for academic and professional growth. It includes creating an online presence, understanding safe internet practices, using collaboration tools, and learning professional communication.

Tasks Summary

Task 1 – Infographic  
Created a digital literacy awareness infographic using Canva. It explains key concepts such as digital tools, online safety, and professional presence.

Task 2 – Digital Portfolio  
Set up professional profiles on platforms like GitHub and LinkedIn to build a strong digital presence.

Task 3 – Platforms  
Completed a beginner coding challenge on HackerRank and created a Google Form quiz for digital literacy awareness.

Task 4 – Email Etiquette  
Drafted professional emails and created a social media do’s and don’ts checklist to understand proper online communication.

Task 5 – Cybercrime Awareness  
Prepared a case study on phishing attacks and created a prevention checklist to promote safe online practices.
